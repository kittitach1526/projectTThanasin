state 0 = แสดงหน้าจอ ชื่อบริษัท+ชื่อwifiที่เคยจำไว้
state 1 = เช็คว่า ssid มีข้อมูลมั้ย ถ้าไม่มีให้ไป ทำ scan wifi และแสดงข้อมูลwifiที่เจอรอบๆบนจอ lcd จากนั้นไป state2
    แต่ถ้ามีข้อมูล ให้รอว่าจะกด A มั้ย ถ้ากดให้ไป state 2
                ถ้าไม่ได้กด A และปล่อยไว้ครบ 5 วิ จะทำการเชื่อมต่อ wifi
state 2 = รอรับค่าจาก keypad ว่าจะเลือกใช้ wifi ssid ตัวไหนแล้วทำการจำใส่ eeprom จากนั้นไป state3
state 3 = เป็นส่วนที่ใช้บันทึกรหัสผ่าน ตอนนี้ รับ Serial

state 4 = เป็นส่วนการเชื่อมต่อ WiFi
state 5 = เป็นส่วนของจอ stanby และรอการแตะบัตร ถ้ามีการแตะบัตรให้ไปที่ state 6
state 6 = เป็นส่วนการติดต่อกับ api
